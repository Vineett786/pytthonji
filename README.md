#  Simple To-Do List (CLI App)

A beginner-friendly command-line To-Do List built in Python.

##  Features

- Add tasks
- List all tasks
- Mark tasks as done
- Delete tasks
- Saves tasks in a `tasks.json` file

##  Usage

```bash
# Add a task
python todo.py add "Buy groceries"

# List tasks
python todo.py list

# Mark task 1 as done
python todo.py done 1

# Delete task 2
python todo.py delete 2
